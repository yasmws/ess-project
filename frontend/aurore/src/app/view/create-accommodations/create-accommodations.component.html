Criar Acomodação

Nome da Acomodação é obrigatório.
Nome da Acomodação deve ter no máximo 20 caracteres.
Localização é obrigatória.
Número de Quartos é obrigatório.
Número de Quartos deve ser um valor inteiro positivo.
Número máximo de hóspedes é obrigatório.
Número máximo de hóspedes deve ser um valor inteiro positivo.
A descrição é obrigatória.
A descrição deve ter no máximo 400 caracteres.
O preço é obrigatório.
